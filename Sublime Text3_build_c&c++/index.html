Sublime Text3_build_c&c++

2020-04-29 评论

Sublime Text 3建立c/c++编译环境

准备工作

  • 下载sublime text3
  • 下载C/C++编译器MinGW

配置环境变量

  • 将gcc所在路径加入环境变量
  • 右键计算机–>属性–>高级系统设置–>环境变量

  • 找到path变量,将gcc安装路径加入进去,gcc安装路径在MinGW/bin文件夹下
  • 配置完成以后,打开cmd窗口,输入gcc -v查看是否安装成功。如果显示gcc版本信息，为设置成功，不然设置失败。

新建sublime text3的C/C++编译系统

  • 第一步，打开sublime text3 选择
    中文版：工具 –> 编译系统 –> 新建编译系统
    英文版：tool –> Build System –> New Build System

  • 输入以下代码保存,发现有很多教程中所贴的代码有一些存在无法无法命令行窗口、中文乱码等问题，故把自己的代码贴出来。

  • 按Ctrl+s保存,文件名为 c.sublime-build 。

  • 保存完之后就能在编译系统中看到C了

    至此，已经配置完了所有环境，快去编个程序试试吧
