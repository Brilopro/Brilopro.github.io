Ubuntu20.4搭建H1ve awd平台

2020-12-15 CTF AWD, CTF 评论

Ubuntu20.4搭建H1ve awd平台

一些依赖包的下载:
sudo apt-get install -y yum-utils device-mapper-persistent-data lvm2 bind-utils
下载docker
sudo apt-get install docker

下载docker.io
``sudo apt-get install docker.io``

验证docker是否安装成功:docker –version

将docker启动并加入开机启动
sudo systemctl start docker #``或``sudo service docker start
sudo systemctl enable docker

安装docker-compose
1.安装epel-release

sudo apt-get install epel-release

2.安装python-pip

sudo apt-get install python3-pip
3.安装docker-compose
sudo apt-get install docker-compose

验证是否安装成功:docker-compose version

下载H1ve
如果没有git需要安装git
sudo apt-get install git

使用git下载h1ve项目
Git clonehttp://github.com/D0g3-Lab/H1ve.git
或者访问http://github.com/D0g3-Lab/H1ve 下载源码

然后进入H1ve目录 然后开启docker
cd H1ve
开启H1ve
sudo docker-compose -f single.yml up &
这个过程是在拉取镜像

关闭命令
Sudo docker-compose -f single.yml down

然后遇到一些错误
错误详情:

解决方案:将docker加入docker用户组
# ``添加docker用户组，一般已存在，不需要执行
sudo groupadd docker
# ``将登陆用户加入到docker用户组中
sudo gpasswd -a $USER docker
# ``更新用户组
newgrp docker
然后重新注销账户重新登录

然后重启H1ve
docker-compose -f single.yml down
docker-compose -f single.yml up
访问本机ip:8000 H1ve至此搭建成功
